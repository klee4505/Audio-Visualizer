DISCOViz: THREE JS WEB AUDIO VISUALIZER
---------------------------------------
- Project By: Kevin Hyun Seok Lee and Esther Tzau
- Project Description: 3D audio visualizer that changes color and pulses to the beat and rhythm of an mp3 file.


Running the code in development
-------------------------------
# Getting Started
Required! Gulp installed `npm install -g gulp`
```
$ cd html-boilerplate
$ npm install
$ npm run dev  
```


# Build to production

```
$ npm run build
```
